# 🗺️ Turkey City Distance Analysis

A C# console application that analyzes distances between 81 cities in Turkey.

## Features

- 📋 Lists all 81 cities with their plate numbers
- 🚗 Random 10-city tour with total distance
- 🏘️ Lists neighboring cities for each province
- 📍 Finds the most distant neighboring city pair
- ✈️ Lists straight-line distances from all cities to İzmir
- 🧭 Finds a route from a random city to İzmir (greedy algorithm)

## Technologies

- C# / .NET 8
- CSV file parsing
- Graph traversal (Greedy)
- Distance matrix (81x81)

## Run

```bash
dotnet run
```

## Dataset

Distance matrix loaded from `ilmesafe.csv` containing road distances between all 81 Turkish provinces.
